# 博士学位论文详细评审结果模板

使用中文输出。若用户要求盲审匿名版本，不写作者姓名、导师姓名、课题组或可识别个人身份的信息，除非这些信息本身是评审对象的一部分且用户明确要求保留。论文中允许陈述博士在读期间论文发表成果、专利、流片或竞赛奖励等学术成果；只要不暴露作者姓名和具体论文名称等可直接识别身份的信息，不应将这类成果陈述本身判定为破坏盲审匿名要求。

## Style Notes

- 优先写“证据 + 判断 + 建议”，少写空泛套话。
- “主要不足”应可执行，例如补充哪类文献、限定哪类结论、增加哪种消融或修复哪类排版问题。
- 对硬件论文的性能结论，要写清基线、工艺、频率、资源、工作负载和指标口径。
- 对安全、可靠性或隐私相关结论，要写清威胁模型和评估边界。
